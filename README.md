# Raindrol AI - 基于 DQN 的自动游戏智能体

本项目使用深度强化学习（DQN）训练一个智能体，能够自动玩一个自制的 Pygame 小游戏（类似“吸血鬼幸存者”风格），通过躲避敌人和释放波纹攻击来获得高分。

## 项目亮点
- 从头实现了一个 **Pygame 游戏环境**，并封装成标准的 Gym 式接口。
- 使用 **PyTorch 实现 DQN 算法**，包括经验回放、目标网络、ε-贪婪探索。
- 精心设计了 **奖励函数** 引导智能体学习有效策略：击杀奖励（+100）、连杀奖励（+50）、蓄力攻击奖励（+10）等。
- 训练 1000+ episode 后，智能体学会了主动攻击和躲避，最高单局击杀 32 个敌人，获得 3500+ 奖励。

## 环境简介
- 游戏窗口 800×600，玩家控制黑色小球，红色方块为敌人。
- 玩家可以移动（8 方向）并按下空格释放波纹攻击（蓄力 1 秒可释放更强波纹）。
- 状态空间：24 维向量（玩家坐标、最近 10 个敌人坐标、蓄力进度、分数）。
- 动作空间：18 个离散动作（9 种移动方向 × 2 种攻击状态）。

## 训练结果
经过约 1000 个 episode 的训练，智能体表现出稳定的策略：
- **平均每局击杀**：5-10 个敌人，最高 32 个。
- **平均每局奖励**：500-1500，最高 3500+。
- **学习曲线**：奖励随训练逐步提升，证明智能体学会了有效策略。

## 如何运行
### 1. 克隆本仓库
```bash
git clone https://github.com/你的用户名/rain-ai-demo.git
cd rain-ai-demo
```
### 2. 安装依赖
确保已安装 Python 3.8+，然后运行：
```bash
pip install -r requirements.txt
```
### 3. 训练模型
```bash
python train.py
```
训练过程会输出每个 episode 的统计信息（步数、攻击次数、击杀数等），并定期保存模型和测试智能体表现。
